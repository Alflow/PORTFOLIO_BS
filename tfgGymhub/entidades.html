Documentación de Gymhub (TFG ALFREDO ALBERRUCHE)

Esta página se ha creado con el propósito de facilitar las imágenes, vídeos o contenido multimedia en general de la memoria del TFG con una mejor resolución y calidad, de manera en que sus textos e información sean legibles.

Entidades de Gymhub

Usuario
  • id: int
  • idRol: int
  • nombre: varchar(255)
  • apellido1: varchar(255)
  • apellido2: varchar(255)
  • email: varchar(255)
  • password: varchar(255)
  • img: varchar(255)
  • dni: varchar(9)
  • activo: tinyint(1)
  • idPlan: int

Relaciones:

  • Tiene múltiples Inscripciones (usuarioclase)
  • Puede tener múltiples Rutinas
  • Un Usuario tiene un UsuarioBiología
  • Un Usuario tiene un UsuarioContacto
  • Un Usuario puede realizar múltiples Pagos
Clase
  • id: int
  • nombre: varchar(50)
  • descripcion: text
  • hora: time
  • dia: varchar(10)
  • aforoMaximo: int
  • caloriasQuemadas: int
  • img: varchar(255)

Relaciones:

  • Tiene múltiples Usuarios (usuarioclase)
Ejercicio
  • id: int
  • nombre: varchar(255)
  • descripcion: text
  • idMusculo: int
  • idTipo: int
  • imagen: varchar(255)

Relaciones:

  • Trabaja múltiples Músculos
  • Clasificado por un TipoEjercicio
  • Incluido en múltiples Rutinas (rutinaejercicio)
Musculo
  • id: int
  • nombre: varchar(255)
  • descripcion: text
  • imagen: varchar(255)

Relaciones:

  • Trabajado por múltiples Ejercicios
Plan
  • id: int
  • nombre: varchar(255)
  • precio: decimal(10,2)
  • descripcion: text
  • img: varchar(255)

Relaciones:

  • Incluye múltiples Clases
Tip
  • id: int
  • texto: text
  • categoria: varchar(50)

Relaciones:

  • Relacionado con múltiples categorías
Inscripcion (usuarioclase)
  • id: int
  • idUsuario: int
  • idClase: int
  • fechaInscripcion: timestamp
  • estado: enum('pendiente','completada')

Relaciones:

  • Relación entre Usuario y Clase
Objetivo
  • id: int
  • nombre: varchar(255)

Relaciones:

  • Relacionado con múltiples Rutinas
Rol
  • id: int
  • nombre: varchar(255)
  • descripcion: varchar(255)

Relaciones:

  • Asignado a múltiples Usuarios
Rutina
  • id: int
  • idUsuario: int
  • nombre: varchar(255)
  • descripcion: text
  • fechaCreacion: date
  • idObjetivo: int

Relaciones:

  • Incluye múltiples Ejercicios (rutinaejercicio)
  • Asignada a un Usuario
RutinaEjercicio
  • idRutina: int
  • idEjercicio: int
  • series: int
  • repeticiones: int
  • orden: int
  • nota: text

Relaciones:

  • Relación entre Rutina y Ejercicio
TipoEjercicio
  • id: int
  • nombre: varchar(255)

Relaciones:

  • Clasifica múltiples Ejercicios
UsuarioBiologia
  • idUsuario: int
  • sexo: tinyint(1)
  • fechaNacimiento: date

Relaciones:

  • Asignado a un Usuario
UsuarioContacto
  • idUsuario: int
  • tlf1: varchar(20)
  • tlf2: varchar(20)
  • direccion: varchar(255)
  • ciudad: varchar(100)
  • cp: varchar(10)

Relaciones:

  • Asignado a un Usuario
